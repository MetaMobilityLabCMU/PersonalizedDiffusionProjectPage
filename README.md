# Personalized Diffusion

This repository contains the code for **Generating Personalized Lower-Limb Kinematics Across Walking Speeds Using Subject-Conditioned Diffusion**.

The website files, including `index.html` and `static/` folder, are used to build and host the project website.

## Model Files
- `PersonalizedDiffusionCode.ipynb`
    Contains the architecture code for the subject-conditioned residual diffusion model.
- `best.pt`
    Trained model checkpoint that is used to load the residual diffusion model.
